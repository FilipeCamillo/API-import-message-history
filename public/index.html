Importar Mensagem

Descrição: Importar mensagem na plataforma.

Parâmetros

Nome Tipo Obrigatório Descrição
companyID string Sim ID da empresa para a qual a mensagem será inserida.
token string Sim Token de autenticação para verificar a validade da empresa.
body string Sim Texto que será gravado
ack string Sim Status de ack da mensagem.
read string Sim Status de leitura da mensagem.
mediaType string Sim Tipo de mídia da mensagem (exemplo: imagem, áudio, vídeo).
mediaUrl string Sim URL da mídia associada à mensagem.
ticketId number Sim ID do ticket associado à mensagem.
createdAt string Sim Data e hora de criação da mensagem.
updatedAt string Sim Data e hora da última atualização da mensagem.
fromMe boolean Sim Indica se a mensagem foi enviada pelo remetente.
isDeleted boolean Sim Indica se a mensagem foi excluída.
contactNumber number Sim Número de telefone DDI+DDD+Número
isPrivate boolean Sim Indica se a mensagem é privada.
isEdited boolean Sim Indica se a mensagem foi editada.

Exemplo de Corpo (Body)

{
     "companyID": "ID da empresa",
     "token": "Token de API",
     "body":"Agora é só fazer a importação",
     "ack": "4",
     "read": true,
     "mediaType": "extendedTextMessage",
     "mediaUrl": "",
     "ticketId": 6,
     "createdAt": "2023-09-10 00:00:31.682 -0300",
     "updatedAt": "2023-09-10 00:00:31.682 -0300",
     "fromMe": 1,
     "isDeleted": false,
     "contactNumber": "5517991565280",
     "isPrivate": false,
     "isEdited": false
 }

Respostas

Código Descrição Exemplo
200 Mensagem inserida com sucesso. 
{
    "message": "Mensagem inserida com sucesso!"
}
400 Número de contato não cadastrado na plataforma. 
{
    "error": "Número de contato não cadastrado."
}
401 Token ou empresa inválidos 
{
    "error": "Token ou empresa inválidos."
}
